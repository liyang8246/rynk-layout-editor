# Rynk Layout Editor — 需求计划

## 产品定位

一个极简的网页端键盘布局编辑器，专注"画布局 + 出 Rynk 配置"，砍掉 KLE 所有花哨功能，只保留核心设计能力。

---

## 技术选型

| 用途 | 方案 | 说明 |
|------|------|------|
| 画布渲染 | HTML div + CSS | absolute positioning，CSS transform 旋转，KLE 原版也是 DOM 渲染 |
| KLE 解析 | @ijprest/kle-serial | KLE 自己用的库，100% 兼容，格式有坑不值得手搓 |
| TOML 输出 | smol-toml | 4.4KB gz，支持序列化+解析，零依赖 |
| 键移动 | 键盘方向键 | 选中后方向键微调，不需要拖拽 |
| 状态管理 | SolidJS createStore | 内置，够用 |
| 快捷键 | 原生 keydown 监听 | Delete/Ctrl+Z/Ctrl+C/V/方向键，几个绑定不值得引库 |
| UI 组件 | daisyUI | 已有，尽量用默认样式 |

外部依赖只加两个：`@ijprest/kle-serial` + `smol-toml`

---

## P1 — 核心画布与键操作

> 能画出一个键盘布局，增删改键

1. **可视化键盘画布** — 在网格上绘制键帽，实时渲染键盘布局
2. **添加键** — 点击画布或工具栏添加新键，默认 1u×1u
3. **删除键** — 选中键后删除
4. **调整大小** — 输入数值或拖拽调整手柄改变键的宽高（支持非整数，如 1.25u、1.5u、2u 等）
5. **移动键** — 方向键微调或输入数值改变键的 x/y 位置
6. **选中与多选** — 点击选中键，支持框选多键
7. **快捷键** — Delete 删除、方向键微调选中键位置

---

## P2 — 形状与矩阵

> 支持异形键，接入矩阵系统

8. **异形键** — 支持第二矩形（x2/y2/w2/h2），用于 ISO Enter、阶梯 Caps 等异形键帽
9. **旋转键** — 设置旋转角度，基于键帽自身中心旋转（与 KLE 的 rx/ry 旋转原点不同，Rynk 只支持中心旋转）
10. **矩阵坐标** — 每个键标注所属的 (row, col) 矩阵位置
11. **矩阵自动编号** — 根据布局自动分配行列号，也支持手动指定
12. **矩阵尺寸** — 显示/设置总行数和列数

---

## P3 — 连线与编码器

> 可视化电气连接，支持旋钮

13. **连线可视化** — 在画布上显示键与矩阵行列的连线关系
14. **编码器** — 在布局中放置旋钮编码器，标注 ID，渲染为 1u 圆形

---

## P4 — 导入导出

> 与外部世界互通

13. **导入 KLE JSON** — 导入 keyboard-layout-editor.com 导出的标准 KLE JSON，自动转换旋转模型（KLE 的 rx/ry 原点旋转 → Rynk 的中心旋转）
14. **导出 Rynk Layout TOML** — 输出符合 rynk-kle 格式的 `[layout]` TOML 配置，包含 map、shapes、variants
15. **导出 KLE JSON** — 输出标准 KLE JSON（保持与原版编辑器的兼容性）

---

## P5 — 布局变体

> 同一把键盘多种布局

16. **变体管理** — 支持多布局变体（如 ANSI vs ISO、整 Backspace vs 分裂 Backspace）
17. **变体切换** — 在不同变体间切换查看
18. **变体中的隐藏键** — 某变体下隐藏的键位，隐藏后后续键自动回流补位
19. **变体中的形状覆盖** — 某变体下使用不同形状的键位（如 2u Backspace 变 1u）

> **Rynk 变体约束**：变体仅影响渲染，不影响键映射行为；隐藏的键必须是基础 map 中存在的键；多选项组不会自动交叉组合，需手动编写；编码器不随变体隐藏或变形。

---

## P6 — 交互打磨

> 提升编辑效率

20. **撤销/重做** — 操作历史支持回退
21. **复制粘贴** — 多选后 Ctrl+C/V 复制粘贴键
